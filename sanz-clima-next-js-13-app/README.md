1. Clone the repo:
   `git clone https://laiaruizm.github.io/SanzClima-full-stack-test`

1. Install NPM packages:
   `npm install`

1. Start the project, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
